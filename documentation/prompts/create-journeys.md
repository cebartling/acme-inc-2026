# Create Journey Prompts


## Create customer profile

```
Create a user journey around
creating a customer profile
within the ACME, Inc. e-commerce platform.
This is meant as an implementation slice through our microservices and user-facing applications.
Think hard about the design of this functionality.
Specific, detailed acceptance criteria are an expectation.
All diagrams should use Mermaid.js. No ASCII art.
Write the content to @documentation/journeys/0001-create-customer-profile.md.
Refer to the epics found in @documentation/epics for roles and responsibilities for each microservice.
Take into account the architecture patterns and standards documented in
@documentation/ARCHITECTURE.md.
Take into account implementation details and technology choices documented in
@documentation/IMPLEMENTATION.md.
Create a feature branch from the `main` branch for this work.
Commit all changes to this feature branch.
When completed, create a PR for merging this feature branch back into
the `main` branch on GitHub.
```
