# Create User Story Prompts


## Infrastructure services

```
Create a user story from @documentation/IMPLEMENTATION.md, Core infrastructure heading.
Write the user story to @documentation/user-stories/0000-infrastructure directory.
Think hard about the implementation of this user story.
Create acceptance criteria for the new user story, where appropriate.
All diagrams should use Mermaid.js. No ASCII art.
Implementation and technology found at @documentation/IMPLEMENTATION.md.
Create a feature branch from the `main` branch for this work.
Commit all changes to this feature branch.
When completed, create a PR for merging this feature branch back into
the `main` branch on GitHub.
```
